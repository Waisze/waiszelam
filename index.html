Accelerometer Pong

A two-player Pong game utilizing two Altera DE0 accelerometers to control each paddle with responsive 1:1 motion. Each DE0 connected to an Altera DE2 board with custom SoC loaded in order to interface the data flow between accelerometers, VGA display and the processor.


This project really sparked my interest in creating software applications for unique hardware such as the Oculus Rift and SoftKinetic depth camera.

Genesys Hackathon

My second hackathon!


Hosted by a Toronto-based telecommunications company, my team was given 24 hours to create an application that utilized their chat API. Our product, named FreeLancr, provided a one-click customer support solution that provided SME and freelancers with a platform in order engage with potential customers using live chat without a dedicated support team.


We aimed to complete a full, marketable product that solved a real-world problem and it paid off. Our team won First Place at the competition, and were rewarded with two Google Glasses!


The project was implemented using NodeJS, MongoDB and Angular. The UI was done through a combination of HTML, CSS and Javascript.
